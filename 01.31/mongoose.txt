mongoose 모듈
몽고디비를 사용하면 하나의 컬렉션 안에 똑같은 속성을 가진 문서 객체를 반복적으로 넣어둔다면 
데이터를 조회할 때도 어떤 속성들이 있는지 미리 알고 있는 상태에서 조회할 수 있습니다. 
관계형 데이터베이스처럼 테이블이나 엑셀시트와 같이 쉽게 데이터를 다룰 수 있도록 제공합니다.

메소드
connect() : mongoose를 사용해 데이터베이스를 연결합니다.
model() : 모델을 정의합니다.
Schema() : 스키마를 정의하는 생성자입니다.

웹브라우저에 웹문서 출력면 웹퍼블리셔,,, 
프론트 엔드란 사용자에게 보여주는 부분을 자바스크립트 언어로 작성
백엔드은 프론트엔드에서 요구하는 데이터를 제공하는 형태를 개발

mysql 모듈
mysql과 연결하기 위한 모듈입니다. mysql 모듈은 커넥션 풀을 사용하여 연결합니다. 데이터베이스 연결 객체가 너무 많이 만들어지는 것을 막고 한번 만든 연결을 재활용할 수 있습니다.

mysql 연결 설정
connectionLimit : 커넥션 풀에서 만들 수 있는 최대 연결 개수를 설정합니다.
host : 연결할 호스트 이름을 설정합니다.
port : 데이터베이스가 사용하는 포트번호를 설정합니다.
user : 데이터베이스 사용자 아이디를 설정합니다.
password : 데이터베이스 사용자 비밀번호를 설정합니다.
database : 데이터베이스 이름을 설정합니다.
debug : 데이터베이스 처리 과정을 로그로 남길지 여부를 설정합니다.
